Code Repository

This repository contains code used in the upstream and downstream analyses for the following publication:
Activity-Driven Remodeling of Genome-Lamina Interactions and DNA Methylation Tunes Neuronal Gene Programs
You will find scripts for processing DamID-seq, WGBS, RNA-seq, and transposable element data, as well as code for LAD/PRG overlap analyses and generating figures and tables.
Contents
The repository is organized into the following analysis packages:

DamID / LAG analysis - DamID preprocessing, DiffBind/GTF annotation, LAG/DEG overlap analysis, and GO enrichment
WGBS / methylation analysis - Differential methylation calling (DMC/DMR), annotation, DEG overlap, and methylation-expression correlation
RNA-seq analysis - Mapping, quantification, differential expression, trajectory clustering, heatmap visualization, and GO enrichment
Transposable element analysis - TE catalog construction, differential expression annotation, TE-gene overlap, IAP summaries, and methylation metaprofiles
Public LAD / PRG overlap analysis - Empirical permutation tests for gene set enrichment in lamina-associated domains
